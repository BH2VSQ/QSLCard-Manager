## ❌ 发现的问题

### 1. 打印功能不符合需求
**需求**：标签打印时要求直接唤起网页的打印请求而不是转换为pdf在网页中打开

**当前实现**：
- PrintQueue.jsx 中使用 `window.open(pdfUrl)` 打开 PDF
- 需要用户手动点击浏览器的打印按钮

**需要修改**：
- 生成 PDF 后自动调用 `window.print()`
- 或者直接生成 HTML 内容并打印，不经过 PDF

### 2. SQL.js 参数类型问题
**问题**：sql.js 对参数类型要求严格，导致 datatype mismatch 错误

**已修复**：
- 修改了 sqljs-adapter.js 的 prepare 方法
- 所有 LIMIT/OFFSET 参数都使用 parseInt()

### 3. 数据库初始化问题
**问题**：callsigns 表中有 null 记录

**已修复**：
- 添加了 WHERE callsign IS NOT NULL 过滤
- 清理了数据库中的 null 记录
