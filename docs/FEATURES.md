# QSL Manager 功能文档

## 📋 功能概览

QSL Manager 是一个完整的业余无线电 QSL 卡片管理系统，提供从日志记录到卡片打印的全流程管理。

## 🎯 核心模块

### 1. 日志管理 (Log Management)

#### 功能特性
- **完整的 QSO 记录**: 支持所有标准字段
  - 基本信息：呼号、日期、时间、频率、波段、模式
  - 信号报告：RST 发送/接收
  - 扩展信息：卫星、中继台、网格坐标
  - QSL 状态：发送/接收状态和日期
- **智能数据验证**: 自动频率到波段转换
- **重复检测**: 防止重复日志条目
- **批量操作**: 支持批量编辑和删除
- **高级过滤**: 按呼号、模式、波段、QSL状态过滤

#### 支持的 QSO 类型
1. **Basic QSO**: 基础通联
2. **Satellite QSO**: 卫星通联
3. **Repeater QSO**: 中继台通联  
4. **Eyeball QSO**: 面对面通联

### 2. QSL 卡片管理 (QSL Card Management)

#### 卡片生成
- **智能卡号生成**: 自动生成唯一 QSL ID
- **双向支持**: 
  - TC (发卡): 确认发送 QSL 卡片
  - RC (收卡): 确认接收 QSL 卡片
- **灵活模式**:
  - 单卡模式: 多条日志共用一个卡号
  - 多卡模式: 每条日志独立卡号

#### 状态管理
- **三阶段流程**: 未分配 → pending → in_stock/out_stock
- **自动状态更新**: 扫码自动更新状态
- **批量处理**: 支持批量状态变更

### 3. 打印系统 (Print System)

#### 标签类型
1. **QSL 标签**:
   - Layout 1 (TC): 6x6网格布局，包含QSO详情和QR码
   - Layout 2 (RC): 单页QR码布局
   - 尺寸: 70mm x 50mm

2. **地址标签**:
   - 发信人标签 (FROM)
   - 收信人标签 (TO)  
   - 智能地址分行
   - 支持电话和国家信息

#### 打印功能
- **单个打印**: 即时打印单个标签
- **批量打印**: 同类型标签合并打印
- **打印队列**: 缓存管理，支持预览
- **自动分页**: 智能处理多页内容

### 4. 出入库管理 (Inventory Management)

#### 扫码功能
- **QR码扫描**: 支持QSL ID扫码
- **状态自动更新**: 扫码即更新出入库状态
- **连续模式**: 支持连续扫码操作
- **详细信息显示**: 显示卡片和关联日志信息

#### 状态跟踪
- **实时状态**: pending → in_stock/out_stock
- **日期记录**: 自动记录状态变更时间
- **呼号信息**: 显示对方和我方呼号

### 5. 地址簿管理 (Address Book)

#### 联系人管理
- **完整信息**: 呼号、姓名、地址、邮编、电话、国家
- **搜索功能**: 按呼号或姓名搜索
- **批量导入**: 支持批量添加联系人
- **地址验证**: 智能地址格式化

#### 地址标签
- **发信人设置**: 默认发信人信息配置
- **收信人选择**: 从地址簿选择或手动输入
- **标签生成**: 自动生成打印就绪的地址标签

### 6. 手动查询 (Manual Search)

#### 查询功能
- **QSL ID查询**: 根据卡片ID查询详细信息
- **卡片详情**: 显示卡片状态、方向、创建时间
- **关联日志**: 显示所有关联的QSO记录
- **实时搜索**: 即时返回查询结果

### 7. 数据导入导出 (Data Import/Export)

#### ADIF 支持
- **导入功能**: 
  - 支持 .adi 和 .adif 文件
  - 自动重复检测
  - 批量导入处理
  - 导入后自动排序
- **导出功能**:
  - 标准 ADIF 格式导出
  - 支持过滤条件导出
  - 兼容主流日志软件

#### 数据迁移
- **Python兼容**: 支持从原Python版本迁移
- **数据完整性**: 保持所有数据和关联关系
- **自动升级**: 数据库结构自动升级
